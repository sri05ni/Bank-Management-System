# Bank Management Application

## Description
Bank Management Application developed using ASP.NET and SQL Server that enables secure user registration and login. Registered users can view account details, transaction history, and perform credit and debit operations for Savings and Joint accounts.

## Technologies Used
- ASP.NET
- SQL Server
  
## Features
- User registration and login
- Savings and Joint account support
- Credit and debit transactions
- Secure authentication

## How to Run
1. Open the project in Visual Studio
2. Configure SQL Server connection
3. Run the application

## Database Setup
- Create a database in SQL Server
- Run the `BankDB.sql` script to create tables and sample data
- Update `Web.config` with your connection string
